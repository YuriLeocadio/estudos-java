# 🥷 Maratona Java - DevDojo

Repositório criado para armazenar todo o progresso, códigos de exemplo e exercícios práticos desenvolvidos durante o curso **Maratona Java (Virado no Jiraya)** ministrado pelo William Suane do DevDojo.

## 🛠️ Tecnologias e Ferramentas Utilizadas

*   **Linguagem:** Java (JDK 25)
*   **IDE:** IntelliJ IDEA
*   **Paradigma principal:** Orientação a Objetos (POO)

## 📌 Conteúdos Estudados

*   [x] Configuração de Ambiente (Instalação do JDK e IDE)
*   [ ] Sintaxe Básica do Java e Estruturas Condicionais
*   [ ] Laços de Repetição e Arrays
*   [ ] Orientação a Objetos (Classes, Métodos, Herança, Polimorfismo)
*   [ ] Tratamento de Exceções
*   [ ] Coleções (Lists, Maps, Sets)
